# Welcome to AIBlog

This is a sample blog post. With **AIBlog**, you can simply drop a Markdown file into your `blog_drafts` directory, and GitHub Actions will automatically generate a polished, comprehensive post like this one using AI.

## Why AIBlog?
- **Frictionless writing**: Just jot down notes, and AI handles the formatting.
- **Beautiful UI**: Built with shadcn/ui and Tailwind CSS.
- **Open Source**: Customize it to your heart's content.
